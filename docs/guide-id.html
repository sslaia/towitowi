Konfigurasikan asisten menulis AI pribadi Anda di TowiTowi

Bila Anda ingin menggunakan Gemini AI untuk membantu merapikan catatan atau membuat postingan media sosial, Anda memerlukan Kunci API Gemini dari Google AI Studio. Berikut cara mendapatkan dan menggunakannya di aplikasi TowiTowi.

1. Login ke AI Studio di Browser

  1. Buka aistudio.google.com
  2. Klik tombol Get started
  3. Login menggunakan akun Gmail Anda
  4. Di dalam AI Studio, cari menu MANAGE di bilah navigasi sebelah kiri
  5. Klik menu Dashboard untuk melihat daftar Kunci API Anda

2. Membuat Proyek TowiTowi

  1. Klik menu Projects di sebelah kiri
  2. Klik tombol Create a new project di sudut kanan atas
  3. Masukkan nama proyek: TowiTowi
  4. Klik tombol Create project

3. Membuat Kunci API

  1. Klik tombol Create API key di sudut kanan atas
  2. Pada formulir yang muncul, masukkan TowiTowi Gemini API key di bidang "Name your key"
  3. Pilih TowiTowi pada bagian dropdown "Choose an imported project"
  4. Klik tombol Create key
  5. Salin Kunci API yang ditampilkan (Anda juga dapat menyalinnya nanti)

4. Menyalin Kunci API

  1. Dari daftar kunci API, klik TowiTowi Gemini API key di kolom "Key"
  2. Kunci API Anda akan terlihat. Klik ikon salin di ujung kanan Kunci API tersebut

5. Memasang Kunci API di Aplikasi TowiTowi

  1. Buka aplikasi TowiTowi Anda
  2. Klik Menu (ikon di sudut kiri bawah)
  3. Pilih Settings (Setelan)
  4. Masukkan Kunci API yang sudah Anda salin sebelumnya pada kolom KUNCI API GEMINI

Tambahan: Personalisasi Gaya Penulisan

Bila Anda ingin asisten AI merapikan tulisan atau membuat draf postingan dengan gaya penulisan Anda sendiri, Anda dapat meminta Gemini AI untuk menyesuaikannya.

Caranya, masukkan petunjuk atau gaya yang Anda inginkan pada kolom INSTRUKSI SISTEM. Kemudian, masukkan contoh tulisan Anda sebelumnya pada kolom ARTIKEL REFERENSI GAYA. Contohnya, salin dan tempel tiga artikel berbeda yang pernah Anda tulis yang mencerminkan gaya khas Anda.

Selamat mencoba dan ber-Towi!
