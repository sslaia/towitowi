Gemini AI Setup Guide

Configure your personal AI writing assistant in TowiTowi

If you want to use Gemini AI to help with restructuring your notes or crafting your social media posts, you need to provide a Gemini API key from Google AI Studio. Here is how you can get it and use it in the TowiTowi app.

1. Login to AI Studio

  1. Go to aistudio.google.com
  2. Click on Get started
  3. Login with your Gmail account
  4. Inside AI Studio, look for MANAGE in the left menu
  5. Click on Dashboard. You will see the list of API keys

2. Create the TowiTowi project

  1. Click on Projects in the left menu
  2. Click Create a new project in the top right corner
  3. Enter the project name: TowiTowi
  4. Click Create project

3. Create the API key

  1. Click on Create API key in the top right corner
  2. In the form, enter TowiTowi Gemini API key in the "Name your key" field
  3. Choose TowiTowi in the "Choose an imported project" dropdown
  4. Click Create key
  5. Copy the API key provided (you can also copy it later)

4. Copy the API key

  1. From the list of API keys, click on TowiTowi Gemini API key in the "Key" column
  2. You will see the API Key. Click on the copy icon at the end of the API key

5. Use the API key in TowiTowi app

  1. Run the TowiTowi app
  2. Click on the Menu (bottom left corner)
  3. Click on Settings
  4. Enter the previously copied API key in GEMINI API KEY

Addition: Style Personalization

If you want to use AI to tidy up your writing and create social media posts, it might be a good idea to have Gemini AI adapt your writing style.

To do this, enter instructions in the SYSTEM INSTRUCTIONS field. Then, enter examples of Reference Articles in the STYLE REFERENCE ARTICLES field. For example, copy three different articles you've written in the past that reflect your own style.

Good luck!
